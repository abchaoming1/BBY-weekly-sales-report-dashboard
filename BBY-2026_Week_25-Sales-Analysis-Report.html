📊 业务线表现深度分析 (Week 25)

核心结论：本周总销量为14,545条，环比+12.88%，同比+53.56%；销售额为$2,641,567，环比+12.68%，同比+69.09%。线上销量4,680条，线上占比32.18%，较上周+6.64pct；POP单店单周11.70，环比+0.31。整体来看，本周增长主要来自重点促销SKU放量与线上渠道回暖。

分业务线表现：骨导线本周8,348条，占比57.39%，环比+24.23%，明显强于大盘；其中S820 / S821系列合计5,926条，环比+36.58%，继续承担主要增量。气导线本周6,197条，占比42.61%，环比+0.52%，整体基本持平，说明本周增长并非全面扩散，而是更集中在少数促销骨干SKU。

耳夹 / 耳挂拆分：耳夹线(E310 / E320 / E210)合计2,310条，环比+0.52%，整体持平；耳挂线(T010 / T511 / T920 / T921 / T910)合计3,887条，环比+0.52%，整体持平。其中E310在促销带动下由534条回升至1,030条，但E320 / E210合计从1,764条回落至1,280条，环比-27.44%，说明新品当前更多是在耳夹内部重新分配需求，尚未形成明显净新增。

关键SKU动态：S820 / S821系列本周5,926条，是本周最强增长引擎，促销条件为4-6天、促销价$124.99、折扣22%。T921本周1,736条，环比+17.06%，在1-6天、促销价$139.99、折扣20%支持下继续放量。E310本周1,030条，环比+92.88%，在4-6天、促销价$139.99、折扣20%推动下，线上销量也由162条升至416条。T010本周无折扣，销量1,117条，环比-14.67%，自然需求仍在，但阶段性被促销SKU分流。S803虽有4天、促销价$89.99、折扣31%，但销量降至899条，环比-18.20%，促销效率明显弱于S820 / T921。

经营判断：本周大盘向上更多来自促销和线上曝光恢复，而不是新品自然扩容。后续需要重点观察两件事：一是S820 / T921促销结束后的留量能力，二是E320 / E210能否摆脱对E310的内部替代，形成真正的耳夹线净新增。
